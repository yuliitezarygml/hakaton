# Fact Guard

Проект по борьбе с дезинформацией и повышению уровня медиаграмотности.

## Описание

Fact Guard — это веб-приложение, построенное с использованием FastAPI и Jinja2, которое предоставляет:

- **Главная страница** — интерактивный элемент, заменяемый через Jinja
- **Библиотека статей** — карточки со статьями, при клике на которые отображается полная информация
- **О нас** — информация о проекте и его миссии

# Основные команды

```bash
sass --watch static/css/style.scss static/css/style.css
```


## Установка

1. **Активировать виртуальное окружение:**

   ```bash
   venv\Scripts\activate
   ```
2. **Установить зависимости:**

   ```bash
   pip install -r requirements.txt
   ```

## Запуск приложения

```bash
python main.py
```

Приложение будет доступно по адресу: `http://127.0.0.1:8000`

Или используйте uvicorn с автоперезагрузкой:

```bash
uvicorn main:app --reload
```

## Переменные окружения

- `USE_FAKE_API_DATA` — включает подстановку фейкового ответа API на странице `/api-response`.
   - `true` (по умолчанию) — данные подставляются без обращения к API.
   - `false` — используется реальный streaming-запрос к API.

## Маршруты

- `GET /` — главная страница
- `GET /library` — библиотека статей
- `GET /article/{article_id}` — страница конкретной статьи
- `GET /about` — страница о нас
